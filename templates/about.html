About This Todo App

This is a simple and clean Flask-based Todo Management Web Application built using Python, HTML, Bootstrap, and SQLite database. It allows users to add, update, delete, and search their daily tasks conveniently.

Features:

  • Add new todos with Title, Description, Status, and Priority.
  • Automatically set Priority to Done if Status is Completed.
  • Update or Delete existing todos.
  • Real-time search functionality by title.
  • Responsive design with clean Bootstrap layout.
  • Date & Time is recorded at the time of todo creation.

Technologies Used:

  • Python 3
  • Flask Framework
  • SQLite Database
  • Jinja2 Templating
  • Bootstrap 5
  • Custom CSS for styling enhancements
